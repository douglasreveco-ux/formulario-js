
# Formulario Web con JavaScript

## Descripción
Este proyecto consiste en el desarrollo de un formulario web que valida datos utilizando JavaScript, eventos DOM y promesas.

El trabajo fue realizado de manera individual como parte de la asignatura Taller de Plataformas Web.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Validación de campos obligatorios
- Validación de correo electrónico mediante RegExp
- Mensajes de error visibles al usuario
- Uso de eventos DOM (submit)
- Simulación de envío con promesas (2 segundos)
- Manejo de errores en el envío

## Autor

## Instrucciones de uso
1. Abrir el archivo `index.html`
2. Completar el formulario
3. Probar validaciones y envío
